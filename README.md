## Stack

HTML / CSS / JS vanilla — aucune dépendance, aucun framework.

- `styles.css` — tout le CSS (partagé entre toutes les pages)
- `main.js` — menu burger, animations au scroll
- Fonts via Google Fonts (Montserrat + DM Sans)
